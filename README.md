# 🎯 Nitro Generator Pro (Tkinter GUI Demo)

Une interface graphique réalisée en **Python (Tkinter)** avec un design inspiré de Discord.

Ce projet est un **prototype éducatif** montrant :
- La création d’une interface moderne en Tkinter
- L’utilisation de threads en Python
- La gestion de logs en temps réel
- Une barre de progression dynamique
- Un système de simulation de vérification de codes

---

## ⚠️ Avertissement

Ce projet est uniquement à but **éducatif et démonstratif**.  
Il ne doit pas être utilisé pour contourner, exploiter ou accéder à des services tiers.

---

## 🚀 Fonctionnalités

- Interface graphique stylisée (thème Discord)
- Génération de codes aléatoires (format simulation)
- Simulation de vérification avec différents statuts :
  - Valide
  - Invalide
  - Déjà utilisé
  - Erreur / Rate limit
- Multi-threading pour simuler des traitements parallèles
- Barre de progression en temps réel
- Logs colorés dans une console intégrée

---

## 🧰 Technologies utilisées

- Python 3
- Tkinter (GUI)
- Threading
- Requests (pour simulation réseau)
- Random / String / Time
