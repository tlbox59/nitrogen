# 🎯 Nitro Generator 


Ce projet est un **prototype éducatif** montrant :
- Un système de simulation de vérification de codes

---

## ⚠️ Avertissement

Ce projet est uniquement à but **éducatif et démonstratif**.  
Il ne doit pas être utilisé pour contourner, exploiter ou accéder à des services tiers.

---

## 🚀 Fonctionnalités


- Génération de codes aléatoires (format simulation)
- Simulation de vérification avec différents statuts

---

## 🧰 Technologies utilisées

- Python 3
- Requests (pour simulation réseau)
- Random / String / Time

---
